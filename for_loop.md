Цикл **`for`** - це спосіб повторювати блок коду кілька разів.  
Він складається з трьох частин:  

1. **Ініціалізація** - задаємо початкове значення (наприклад, `int i = 0;`).  
2. **Умова** - перевіряємо, чи треба ще продовжувати цикл (`i < 10`).  
3. **Крок** - змінюємо значення після кожної ітерації (`++i`).

Якщо умова є правдивою, код всередині циклу виконується; потім йде крок, а потім знову перевірка умови.

---

### Коли використовувати

| Ситуація | Чому `for` підходить |
|----------|----------------------|
| Перебір масиву або вектору | Легко задавати індекс і дати кінець. |
| Підрахунок кількості елементів | Відомо, скільки разів треба виконати цикл. |
| Генерація послідовності чисел | Наприклад, від 1 до 100. |
| Обробка рядків по символам | Індекс `i` – позиція в рядку. |

---

### Приклади
1. Вивести числа від 0 до 9
```cpp
for (int i = 0; i < 10; ++i) {
    std::cout << i << ' ';
}
std::cout << '\n';
```

2. Підрахунок сумарної довжини рядків у векторі
```cpp
std::vector<std::string> words = {"cpp", "rocks", "openai"};
int totalLength = 0;
for (size_t i = 0; i < words.size(); ++i) {
    totalLength += words[i].length();
}
std::cout << "Total length: " << totalLength << '\n';
```

3. Перебір масиву і виведення парних чисел
```cpp
int arr[] = {1,2,3,4,5,6};
for (size_t i = 0; i < sizeof(arr)/sizeof(arr[0]); ++i) {
    if (arr[i] % 2 == 0)
        std::cout << arr[i] << ' ';
}
std::cout << '\n';
```
