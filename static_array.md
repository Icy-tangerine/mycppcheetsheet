Статичний масив — це структура даних, яка зберігає фіксовану кількість елементів одного типу в безперервній області пам'яті. Розмір масиву має бути відомий на етапі компіляції.

---

### Коли використовувати

| Ситуація | Чому підходить |
|----------|----------------|
| Відома кількість елементів | Якщо розмір даних не змінюється під час роботи програми. |
| Швидкий доступ | Доступ до будь-якого елемента за індексом відбувається за константний час $O(1)$. |
| Економія пам'яті | Немає додаткових витрат на керування пам'яттю, як у динамічних структурах. |

---

### Приклади

#### 1. Оголошення та ініціалізація
```c++
#include <iostream>
#include <ctime>
#include <cstdlib>

int main() {
    int a[3] = {1, 2, 3}; // Оголошення та ініціалізація
    int b[5];             // Оголошення без ініціалізації

    srand(time(0));
    
    // Заповнення та виведення
    for(int i = 0; i < 5; i++) {
        b[i] = rand() % 100;
        std::cout << "b[" << i << "] = " << b[i] << "\n";
    }
    return 0;
}
```

#### 2. Багатовимірні масиви
```c++
#include <iostream>

int main() {
    // Масив 2x3 (2 рядки, 3 стовпці)
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 3; j++) {
            std::cout << matrix[i][j] << " ";
        }
        std::cout << "\n";
    }
    return 0;
}
```

#### 3. Масиви з автоматичним визначенням розміру
```c++
int arr[] = {10, 20, 30, 40}; // Розмір буде 4 автоматично
```
