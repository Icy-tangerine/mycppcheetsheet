Оператор **`delete`** використовується для звільнення пам'яті, яка була виділена динамічно за допомогою оператора `new`. Це критично важливо для запобігання витокам пам'яті.

---

### Як це працює
1. Викликається **деструктор** об'єкта (якщо це клас).
2. Пам'ять повертається системі (стає доступною для інших частин програми).

---

### Варіанти використання

| Синтаксис | Коли використовувати |
|-----------|----------------------|
| `delete ptr;` | Для одиничних об'єктів, виділених через `new`. |
| `delete[] ptr;` | Для масивів, виділених через `new[]`. |

> **Важливо**: Використання неправильної форми (наприклад, `delete` замість `delete[]` для масиву) призводить до невизначеної поведінки (undefined behavior).

---

### Приклад

```cpp
#include <iostream>

int main() {
    // Виділення пам'яті
    int* num = new int(5);
    int* array = new int[5];

    // Заповнення масиву
    for(int i = 0; i < 5; i++)
        array[i] = 5 - i;

    // Виведення даних
    std::cout << "num = " << *num << "\n";
    std::cout << "array: ";
    for(int i = 0; i < 5; i++)
        std::cout << array[i] << " ";
    std::cout << std::endl;

    // Вивільнення пам'яті
    delete num;
    delete[] array;

    return 0;
}
```
