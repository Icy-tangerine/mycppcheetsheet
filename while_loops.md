# while_loop

Цикл while повторює блок коду, поки умова залишається правдивою (true).  
Якщо умова спочатку хибна – код всередині не виконується.

> **Пам’ятайте:** перед виконанням циклу треба ініціалізувати змінну, що використовується в умові.

---

## Коли його використовувати  

- коли кількість ітерацій невідома заздалегідь  
- коли потрібно чекати певної події (наприклад, введення користувача)  
- при читанні даних до кінця файлу чи потоку  
- у простих задачах: підрахунок, пошук, очікування

---

## Приклади коду  

```cpp
// 1. Підрахунок від 1 до 5
int i = 1;
while (i <= 5) {
    std::cout << i << " ";
    ++i;          // важливо змінювати i, інакше цикл буде безкінечним
}
```

```cpp
// 2. Читання чисел до введення 0
int n;
std::cin >> n;
while (n != 0) {
    std::cout << "Ви ввели: " << n << '\n';
    std::cin >> n;   // оновлюємо значення
}
```

```cpp
// 3. Читання рядка до EOF (кінець файлу)
std::string line;
while (std::getline(std::cin, line)) {
    std::cout << "Рядок: " << line << '\n';
}
```

# do_while_loop

Цикл do_while - це цикл з післяумовою, він виконує блок коду, після чого, перевіряє умову, якщо умова залишається правдивою, блок коду виконується повторно. Якщо умова спочатку хибна - код всередині виконується один раз.

# Цикл `do_while`

## Коротке пояснення  
`do_while` – це цикл **з післяумовою**.  
1️⃣ Спочатку виконується блок коду.  
2️⃣ Потім перевіряється умова.  
3️⃣ Якщо умова **істина**, код повторюється.  
4️⃣ Якщо умова спочатку **хибна**, код виконається **один раз**.

> **Пам’ятайте:** `do_while` гарантовано виконує тіло циклу хоча б один раз.

---

## Коли використовувати

| Ситуація                                                         | Чому підходить                             |
| ---------------------------------------------------------------- | ------------------------------------------ |
| Потрібно запустити код, а потім перевіряти умову                 | Тіло буде виконано принаймні один раз      |
| Введення користувача з перевіркою після введення                 | Перевірка відбувається після кожного вводу |
| Повторювати дію до тих пір, поки не буде вказана конкретна умова | Легко читати і зрозуміло                   |

---

## Приклади

```cpp
// 1. Введення числа, доки воно не стане > 0
int n;
do {
    std::cout << "Введіть число > 0: ";
    std::cin >> n;
} while (n <= 0);
```

```cpp
// 2. Підрахунок сумарної кількості введених цифр, поки користувач не натисне 'q'
char ch;
int sum = 0;
do {
    std::cout << "Введіть цифру або q для виходу: ";
    std::cin >> ch;
    if (ch >= '0' && ch <= '9')
        sum += ch - '0';
} while (ch != 'q');
std::cout << "Сума цифр = " << sum << '\n';
```

```cpp
// 3. Перевірка парності числа, хоча б один раз
int x = 7;
do {
    std::cout << x << " не парне\n";
    ++x; // змінюємо число, щоб вийти з циклу
} while (x % 2 == 0);
```
