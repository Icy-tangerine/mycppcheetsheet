по плану його потрібно відкривати з допомогою [obsidian](https://obsidian.md)

скачати репозиторій можна з допомогою [git](https://git-scm.com/install/windows)

командою
``` bash
git clone https://github.com/Icy-tangerine/mycppcheetsheet.git
```

оновлювати репозиторій можна командою 
```bash
git pull
```

хоч ви й неповинні його оновлювати, мною заплановано щоб **ви** розширювали його **самостійно** , щоб ви зробили його **своїм**
