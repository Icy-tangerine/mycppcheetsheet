Функція — це іменований блок коду, який виконує певне завдання. Функції дозволяють розбивати програму на логічні модулі, що полегшує читання та підтримку коду.

---

### Структура функції
Вона складається з:
- **Типу повернення** — тип даних, які функція повертає (або `void`, якщо нічого).
- **Назва** — унікальне ім'я для виклику.
- **Параметри** — вхідні дані (можуть бути відсутні).
- **Тіло** — код, який виконується.

```cpp
тип_повернення назва_функції(параметри) {
    // тіло функції
    return значення; // якщо тип не void
}
```

---

### Коли використовувати

| Ситуація | Перевага |
|----------|----------|
| **Повторювані дії** | Уникаємо дублювання коду. |
| **Організація логіки** | Програма стає зрозумілішою та структурованою. |
| **Тестування** | Легше перевірити окрему частину програми. |
| **Модульність** | Функції можна виносити в окремі файли та бібліотеки. |

---

### Приклади

#### 1. Функція без параметрів і повернення
```cpp
#include <iostream>

void sayHello() {
    std::cout << "Привіт!\n";
}

int main() {
    sayHello(); // Виведе "Привіт!"
    return 0;
}
```

#### 2. Функція з параметрами та поверненням значення
```cpp
#include <iostream>

int square(int x) {
    return x * x;
}

double average(double a, double b, double c) {
    return (a + b + c) / 3.0;
}

int main() {
    int val = square(5); // val == 25
    double avg = average(1.2, 3.4, 5.6);
    std::cout << "Square: " << val << ", Average: " << avg << "\n";
    return 0;
}
```
