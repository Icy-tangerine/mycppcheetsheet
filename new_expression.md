Оператор **`new`** використовується для динамічного виділення пам'яті в купі (heap) під час виконання програми. Він повертає вказівник на виділену область пам'яті.

---

### Як це працює
1. Обчислюється обсяг пам'яті, необхідний для об'єкта або масиву.
2. Виділяється пам'ять у купі.
3. Викликається **конструктор** (якщо це об'єкт класу).
4. Передається адреса (вказівник) на початок виділеної пам'яті.

---

### Варіанти використання

| Синтаксис | Опис |
|-----------|------|
| `new T;` | Виділяє пам'ять для одного об'єкта типу `T`. |
| `new T(args);` | Виділяє пам'ять та ініціалізує об'єкт аргументами. |
| `new T[N];` | Виділяє пам'ять для масиву з `N` елементів типу `T`. |

---

### Приклад

```cpp
#include <iostream>

int main() {
    // 1. Виділення пам'яті для одного числа
    int* pNum = new int(42);
    std::cout << "Value: " << *pNum << "\n";

    // 2. Виділення пам'яті для масиву
    int size = 5;
    int* pArray = new int[size];
    for (int i = 0; i < size; ++i) {
        pArray[i] = i * 10;
    }

    // Обов'язкове вивільнення пам'яті
    delete pNum;
    delete[] pArray;

    return 0;
}
```
