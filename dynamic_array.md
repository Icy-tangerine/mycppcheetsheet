Динамічний масив — це масив, розмір якого визначається під час виконання програми (runtime). На відміну від статичних масивів, пам'ять для них виділяється в **купі** (heap).

---

### Коли використовувати

| Ситуація | Перевага динамічного масиву |
|----------|-----------------------------|
| **Розмір невідомий заздалегідь** | Можна запитати розмір у користувача або обчислити його під час виконання. |
| **Великі обсяги даних** | Купа має значно більше місця, ніж стек, де зберігаються статичні масиви. |
| **Зміна розміру** | Можна перестворити масив з новим розміром або використати `std::vector`. |

---

### Приклад використання

```cpp
#include <iostream>
#include <cstdlib>

int main() {
    size_t size = 0;
    std::cout << "Enter size of array: ";
    std::cin >> size;

    // Виділення пам'яті (allocating memory)
    int* array = new int[size];

    // Запис у пам'ять
    for(size_t i = 0; i < size; i++)
        array[i] = rand() % 100;

    // Читання з пам'яті
    for(size_t i = 0; i < size; i++)
        std::cout << array[i] << " ";
    std::cout << std::endl;

    // Вивільнення пам'яті (releasing memory)
    delete[] array;
    
    return 0;
}
```

---

### Пояснення
1. **`new int[size]`**: Виділяє блок пам'яті в купі. Адреса першого елемента зберігається у [[pointer|вказівнику]].
2. **`delete[]`**: Обов'язковий крок для запобігання витокам пам'яті. Використовується саме квадратні дужки `[]`, оскільки ми видаляємо масив.
