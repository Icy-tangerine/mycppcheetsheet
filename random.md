Генерація випадкових чисел у C++ може здійснюватися за допомогою класичних функцій C (`rand`) або сучасних засобів бібліотеки `<random>`.

---

### Коли використовувати

| Метод | Ситуація | Опис |
|-------|----------|------|
| `rand()` | Прості задачі | Швидка генерація псевдовипадкових чисел. Потребує `srand` для ініціалізації. |
| `%` (остача) | Обмеження діапазону | Дозволяє отримати число в межах `[0, N-1]`. |
| `srand(time(0))` | Ініціалізація | Встановлює "зерно" (seed) на основі поточного часу, щоб числа були різними при кожному запуску. |

---

### Приклади

#### 1. Базова генерація (rand)
```c++
#include <iostream>
#include <cstdlib> // для rand() та srand()
#include <ctime>   // для time()

int main() {
    // Встановлення зерна
    srand(time(0));

    for(int i = 0; i < 5; i++) {
        // Виведення випадкового числа від 0 до RAND_MAX
        std::cout << rand() << std::endl;
    }
    return 0;
}
```

#### 2. Генерація в діапазоні
Щоб отримати число в діапазоні `[min, max]`, використовується формула:
`min + rand() % (max - min + 1)`.
```c++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
    srand(time(0));
    
    // Діапазон від 10 до 60
    int min = 10;
    int max = 60;

    for(int i = 0; i < 5; i++) {
        int random_num = min + rand() % (max - min + 1);
        std::cout << random_num << std::endl;
    }
    return 0;
}
```

#### Генерація з std::uniform_int_distribution

```c++
#include <iostream>
#include <random>
 
int main()
{
    std::random_device rd;  // a seed source for the random number engine
    std::mt19937 gen(rd()); // mersenne_twister_engine seeded with rd()
    std::uniform_int_distribution<> distrib(1, 6);
    // Use distrib to transform the random unsigned int
    // generated by gen into an int in [1, 6]
    for (int n = 0; n != 10; ++n)
        std::cout << distrib(gen) << ' ';
    std::cout << '\n';
}
```
