Цикли дозволяють виконувати блок коду багаторазово. У C++ основними є цикли `while`, `for` та `do-while`.

---

### Коли використовувати

| Цикл       | Ситуація                    | Чому підходить                                                   |
| ---------- | --------------------------- | ---------------------------------------------------------------- |
| `while`    | Невідома кількість ітерацій | Виконується, поки умова істинна. Зручно для очікування події.    |
| `for`      | Відома кількість ітерацій   | Ідеально для перебору масивів, контейнерів або діапазонів чисел. |
| `do-while` | Гарантоване виконання       | Тіло циклу виконається хоча б один раз перед перевіркою умови.   |

---

### Приклади

#### 1. Цикл [[while_loops|while]]
Використовується, коли потрібно повторювати блок коду невідому наперед кількість разів.

```c++
#include <iostream>
int main(){
	int i=0; // ініціалізація змінної
	while(i < 10){ // перевірка умови
		i++; // підвищення значення змінної на 1
		std::cout << i << std::endl; // виведення значення змінної
	} // перехід до умови відбувається автоматично
}
```

#### 2. Цикл [[for_loop|for]]
Зазвичай використовується, коли потрібно ітеруватись по контейнеру даних чи по значеннях функції.

```c++
#include <iostream>
#include <cmath>
int main(){
	for(float i = 0; i < 3.14; i += (3.14159265 / 10))
	{
		std::cout << sin(i) << std::endl;
	}
}
```
